### Deploying a contract to Polygon
1. Set up metamask to connect to the test network
   
  Network Name: Polygon zkEVM Testnet
  
  RPC URL: https://rpc.cardona.zkevm-rpc.com
  
  Chain ID: 2442
  
  Currency Symbol: ETH
  
  Block Explorer URL: https://www.oklink.com/polygon-zkevm-test
  
2. Transfer some test ETH to your account on polygon zkEVM Testnet using the bridge [bridge interface](https://public.zkevm-test.net/)
  
3. Create a simple contract in Remix, and deploy using metamask as the Injected provider.

4. Check your deployment and transactions using the block explorer
